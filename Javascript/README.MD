# 📚 JavaScript-Estudo

Repositório criado para registrar minha evolução no aprendizado de JavaScript.

Aqui eu salvo exercícios, projetos e anotações feitas durante os estudos.

## 📂 Estrutura do projeto

* 📖 aprendizado → imagens e resumos do que aprendi
* 🧠 exercicios → exercícios de lógica em JavaScript
* 💻 projetos → pequenos projetos feitos no console
* 📦 node_modules → dependências do Node.js
* package.json → configuração do projeto Node

## 🚀 Tecnologias usadas

* JavaScript
* Node.js
* readline-sync

Aprender JavaScript do zero até nível avançado,
com foco em backend e cloud computing.

Este repositório será atualizado conforme eu evoluir nos estudos.
