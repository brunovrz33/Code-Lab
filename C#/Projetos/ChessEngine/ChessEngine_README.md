# ♟️ Chess Engine

> Jogo de xadrez desenvolvido em C# para rodar no terminal.

---

## 📌 Sobre

Projeto desenvolvido durante o curso do professor **Nélio Alves** na Udemy, com incremento próprio de uma funcionalidade de **timer por jogada**.

O jogo roda completamente no terminal, com toda a lógica do xadrez implementada em C#, incluindo movimentos especiais, verificação de xeque e xeque-mate.

---

## ✨ Funcionalidades

- ♟️ Jogo de xadrez completo no terminal
- ✅ Movimentos válidos para todas as peças
- 👑 Detecção de xeque e xeque-mate
- ⏱️ **Timer por jogada** *(funcionalidade adicionada por mim)*

---

## 🛠️ Tecnologia

- **Linguagem:** C#
- **Plataforma:** .NET
- **Interface:** Terminal

---

## ▶️ Como executar

1. Clone o repositório
2. Acesse a pasta do projeto
3. Execute:

```bash
dotnet run
```
